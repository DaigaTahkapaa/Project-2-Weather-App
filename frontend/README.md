In this project, you will use AJAX to fetch data from a live REST API, process it, and display it dynamically on a web page.

Weather Forecast APP Using Live REST API from OpenWeatherMap
Implement a web page where the user can enter a city name to get current weather and forecast
The page then shows temperature, conditions, icons, and 5-day forecast
Display additional info like humidity, wind speed, etc.
Create nice layout and style for the page
Extra points for geolocation integration or unit toggle (C/F)
OpenWeatherMap API is at: https://openweathermap.org/apiLinks to an external site.

As the  OpenWeather API requires Latitude and Longitude for making API calls and additional API - Geocoding API - was used to convert specified city into the exact geographical coordinates
